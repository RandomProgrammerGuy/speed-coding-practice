## Swift

### Lightiary

![Static Badge](https://img.shields.io/badge/status-in_progress-orange?style=for-the-badge)

Lightiary is a lightweight, minimal diary application that allows users to add, edit, remove and view diary entries on their iPhone or iPad. The user interface is designed to be simple and intuitive.

**Features**

- The ability to add, remove, view and save diary entries. Each entry has a title and some text. The creation date and last modification date are stored for each entry
- The user can sort their diary entries by creation date, modification date or title (alphabetical order)
