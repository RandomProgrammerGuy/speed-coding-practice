# Speed-coding practice repository

This repository includes small programs that I have made in a short timeframe (in most cases one or two days) to practice speed-coding and to be more comfortable with developement enviornments. **NOT ALL OF THE CODE IN THIS REPOSITORY WILL WORK! Some code is untested and sloppy. Use at your own risk!** All of the code in this repository is under the MIT License.
